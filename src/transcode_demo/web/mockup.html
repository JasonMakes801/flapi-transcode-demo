What this is

A reference integration: drive the FilmLight render API (flapi) from your own app to mass-transcode camera originals to editorial proxies, with a small web GUI on top. The point is that the API is yours to build on.

The stack — deliberately tiny

Backend
Python + FastAPI (one process)
Frontend
Alpine.js, served by FastAPI — no Node, no npm, no build step
Render
FilmLight flapi; each of 3 workers launches its own child flapid
Dependencies
fastapi, uvicorn, and the Baselight filmlightapi wheel
Run
./bootstrap.sh once, then ./run.sh → http://localhost:8080
